## Important Note: This Creates a Deal, Not a Registration

**⚠️ Clarification Needed:**

This SOP creates a **Deal**, not a **Registration Record**. These are different:

| Record Type | Purpose | Module |
|-------------|---------|--------|
| Deal | Tracks sales opportunity, quotes, invoices | Deals |
| Registration Record | Tracks student attendance, compliance, certification | Registration_Records |

To fully register a student, you may also need to create a Registration Record in the Course's Registration Records section.

## Corrected SOP Summary

### Recommended Updated Procedure

**Step 1: Locate Course**
1. Go to Courses module
2. Use filters (location, date, course name)
3. Open the correct course record

**Step 2: Create Deal**
1. In Course record, go to Deals section
2. Click "..." > New Deal
3. Set required fields:
   - Contact_Name (student)
   - Account_Name (company/payer)
   - Courseaa (auto-set if creating from Course)
4. Do NOT enter Deal Name - it auto-generates
5. Save the Deal

**Automatic on Save:**
- Deal_ID assigned
- Deal_Name generated (P - [Company] - [Location] - [Course])
- Amount calculated from Course pricing

**Step 3: Complete Contact Details**
1. Open the linked Contact record
2. Verify Email and Mobile are populated
3. Add if missing (required for notifications)
4. Add notes to Deal Description if needed

**Step 4: Confirm**
1. Return to Course record
2. Verify Deal appears in Deals list
3. Student is now registered for quoting

**Note:** To track attendance and certification, also create a Registration Record.
