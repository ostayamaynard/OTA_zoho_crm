# Registration Process

## SOP: Registering a Student in the CRM for a Public Course

### Step 1: Locate the Course in CRM
- Go to the Courses module in Zoho CRM.
- If the tab is not visible, click the three dots in the top bar and find it in the dropdown.
- Use filters to locate the course:
  - Filter by location (e.g., "Mackay") or
  - Filter by month (e.g., "July") using the course name field.
- Select the correct course you want to register the student into (e.g., Part B course).

### Step 2: Create a New Deal for the Student
- Scroll to the Deals section in the course record and click the three dots > New Deal.
- Fill in the deal name in the format: CourseCode + Location + Date + Student/Company Name.
  - Example: TAE22B Mackay July Smith
- If applicable, select or create the associated Account (company) for the student.

### Step 3: Complete Student Details
- Ensure the Contact record for the student is linked to the Deal.
- Add or confirm:
  - Email
  - Mobile number
  - Any special notes relevant to the course.

### Step 4: Save and Confirm Registration
- Save the Deal.
- Confirm the student appears in the course's related Deals list.

The student is now registered in the CRM for the selected public course.
